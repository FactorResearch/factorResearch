# Factor Research Design Bible

Version: 1.0

This folder is the single source of truth for every visual and interaction decision
inside Factor Research.

## Rules

Codex must always read documents in this order:

1. DESIGN_SYSTEM.md
2. COMPONENT_LIBRARY.md
3. IMPLEMENTATION_RULES.md
4. RESPONSIVE.md
5. ACCESSIBILITY.md
6. ANIMATIONS.md
7. SPEC.md for the page being implemented
8. prototype/<page>.html

Never redesign.

Never simplify.

Never remove information.

Never create new components unless instructed.

If anything conflicts:

DESIGN_SYSTEM.md wins.

If HTML differs from SPEC.md,

SPEC.md wins.

If uncertainty exists,

ask before implementing.

## Development Philosophy

Factor Research is a professional financial research platform.

Every decision should increase:

• Trust

• Information density

• Speed

• Consistency

• Accessibility

Never sacrifice financial information for aesthetics.

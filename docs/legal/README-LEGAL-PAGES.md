# Cenvarn Legal Pages

This folder contains a launch-ready legal-page set for **Cenvarn**, a web-based quantitative investment research and portfolio-analysis platform.

## Publish at launch

1. `privacy-policy.md`
2. `terms-of-service.md`
3. `financial-and-investment-disclaimer.md`
4. `cookie-policy.md`
5. `acceptable-use-policy.md`
6. `refund-and-cancellation-policy.md`
7. `accessibility-statement.md`
8. `security-and-responsible-disclosure.md`

## Add later when applicable

- A **Data Processing Addendum (DPA)** if you sell to businesses or process personal data for enterprise customers.
- A **Subprocessor List** once your production vendors are finalized.
- A **California Privacy Notice** if California privacy-law thresholds or obligations apply.
- An **EU/UK Privacy Addendum** and cookie consent system before intentionally marketing to EU/UK users.
- A **Do Not Sell or Share My Personal Information** page only if your practices trigger that requirement.
- An **API Terms of Use** page before offering an API.
- A **Community Guidelines** page before launching comments, user posts, shared strategies, or public profiles.
- A **Copyright/DMCA Policy** if users may upload or publish content.
- A **Contest Rules** page for each giveaway or promotion.
- An **Affiliate Disclosure** before using affiliate links.
- An **Enterprise Service Agreement and SLA** before selling negotiated business plans.

- `[BUSINESS MAILING ADDRESS]`
- Payment processor name
- Analytics and monitoring providers
- Hosting and database providers
- Actual retention periods
- Governing province and court location
- Minimum user age
- Subscription billing and refund rules

## Implementation notes

- Put persistent links to Privacy, Terms, Disclaimer, Cookies, Accessibility, and Security in the website footer.
- Require affirmative acceptance of the Terms and Privacy Policy when users create an account.
- Store the policy version, timestamp, user ID, and acceptance method.
- Obtain separate, optional consent for marketing emails. Do not bundle marketing consent into account creation.
- Display the investment disclaimer near scores, forecasts, intrinsic-value estimates, backtests, portfolio tools, and custom factor models.
- Show material assumptions, data dates, model limitations, and source limitations beside analysis results.
- Have a Canadian technology/privacy lawyer review these drafts before launch, especially the subscription, liability, securities-law, and arbitration provisions.
